⚙ Admin — Bazar de mudança

Edite o catálogo e publique direto no site
Como usar:
  1. Cole seu token uma vez (cartão abaixo) — fica salvo só neste navegador.
  2. Clique ⟳ Carregar do site para pegar a versão mais recente.
  3. Adicione/edite produtos. Para fotos novas, use Enviar fotos dentro do produto.
  4. Clique ⬆ Publicar no site. O site atualiza sozinho em ~1 minuto.

Conexão sem token

O token é a sua credencial para publicar. Ele fica guardado apenas neste navegador (localStorage) e nunca é enviado para outro lugar além do GitHub. Para revogar o acesso, basta apagar o token no GitHub.
