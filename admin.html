⚙ Admin — Bazar de mudança

Ferramenta de edição do catálogo · gera o produtos.js
Como usar:
  1. Adicione fotos novas na pasta img/ (crie uma subpasta por produto).
  2. Edite aqui: adicione, exclua, reordene ou marque como vendido.
  3. Clique em ⬇ Baixar produtos.js e substitua o arquivo na pasta do site.
  4. Suba para o GitHub (ou arraste a pasta no Netlify) — o site atualiza.
